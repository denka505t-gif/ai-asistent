# Fishing Assistant — прогресс

## Задача
Создать Telegram-бота помощника рыболова.

## Этапы

- [x] 1. Скаффолд проекта (папка, package.json, структура)
- [x] 2. База рыб (data/fish.json) — 15 видов (щука, окунь, судак, жерех, голавль, язь, форель, хариус, карп, лещ, сом, налим, линь, карась, плотва)
- [x] 3. Сервис OpenStreetMap — поиск водоёмов по названию и GPS
- [x] 4. Сервис OpenWeatherMap — погода
- [x] 5. Сервис Claude API — AI-рекомендации
- [x] 6. Парсер рыболовных форумов (fishx.org, ribak.ru)
- [x] 7. Telegram-бот: хэндлеры сценариев А и Б
- [x] 8. env.example
- [ ] 9. Создать GitHub-репозиторий и запушить
- [ ] 10. Настроить .env и запустить бота
