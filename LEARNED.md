## 2026-07-13 — Пустые ответы бота = истёкшая авторизация Claude
Контекст: бот на VPS отвечал «(пустой ответ)», процесс не падал, ошибок не было
Вывод: первым делом проверять OAuth-авторизацию claude CLI (knowledge/claude-reauth-manual.md), а не искать баг в коде бота
Применение: при любых «тихих» отказах бота

## 2026-07-15 — «WebSocket 1006» в VS Code — не всегда сеть
Контекст: VS Code не подключался к туннелю; сервер был жив, интернет работал
Вывод: зависшее автообновление VS Code ломает его внутренние соединения; проверять процессы CodeSetup и совпадение версий раньше, чем сеть и VPN. Обходной путь — Remote-SSH
Применение: при ошибках подключения к удалённому серверу из VS Code

## 2026-07-15 — Каждый SSH-ключ в authorized_keys — на своей строке
Контекст: дописанный ключ приклеился к строке ключа Beget (без перевода строки в конце) — сервер отвергал вход
Вывод: после дописывания ключа проверить файл: один ключ = одна строка
Применение: при настройке SSH-доступа на любой сервер
